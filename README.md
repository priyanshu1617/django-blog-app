# Django Blog Application

A full-featured blog application built using Django, inspired by the book **Django 5 by Example**.

---

## Features

- User authentication (login/logout)
- Create, update, delete blog posts
- Comments system
- Tagging system
- SEO-friendly URLs
- Pagination

---

## Tech Stack

- Python
- Django
- SQLite
- HTML, CSS

---

## ⚙️ Installation

1. Clone the repository:
```bash
git clone https://github.com/priyanshu1617/django-blog-app.git
```

2. Navigate into the project:
```bash
cd django-blog-app
```

3. Create virtual environment:
```bash
python -m venv env
source env/bin/activate   # Windows: env\Scripts\activate
```

4. Install dependencies:
```bash
pip install -r requirements.txt
```

5. Run migrations:
```bash
python manage.py migrate
```

6. Start server:
```bash
python manage.py runserver
```
